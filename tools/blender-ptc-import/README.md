# Blender PTC import

[Русская версия Readme](README_RUS.md)

## Blender Plugin Installation
```
Edit -> Preferences... -> Add-ons -> Install... -> Select import_ptc.py from io_import_ptc folder -> Enable "Import: SeaDogs PTC"
```

## Blender Usage
```
File -> Import... -> PTC Import(.ptc)
```
