# Blender PTC import

## Установка плагина в Blender
```
Edit -> Preferences... -> Add-ons -> Install... -> Выбрать import_ptc.py из папки io_import_ptc -> Включить "Import: SeaDogs PTC"
```

## Использование плагина в Blender
```
File -> Import... -> PTC Import(.ptc)
```
